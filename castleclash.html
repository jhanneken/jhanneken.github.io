Castle Clash

Studienprojekt

Zusammen mit einem Kommilitonen habe ich ein Geschicklichkeitsspiel namens Castle Clash entwickelt, das verschiedene Spielmodi enthält. Zu diesen Spielmodi gehören ein Jump’n Run- ein Verteidigungs- und ein Münzsammelspiel.

Das Spiel ist in Processing, einer auf Java basierenden Programmiersprache, geschrieben.
